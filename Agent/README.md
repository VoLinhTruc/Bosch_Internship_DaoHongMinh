# Gemini File Agent

AI agent to read and write files in `./workspace`.

## Setup

From the `Agent` folder:

```powershell
.venv\Scripts\activate       
python -m pip install -r requirements.txt
```

Add API key to `.env`:

```text
GEMINI_API_KEY=key
```

## Run

With the virtual environment activated:

```powershell
python main.py
```
